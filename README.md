# 任意门 / Anywhere Door

> 曾用名：浮生 / The Reveries

一个移动优先的纯网页 **AI 活体文字世界**：像刷抖音一样竖滑发现"世界"，推开一扇门就踏入另一个认得你的世界——长驻沉浸；一个 **God 引擎**维护结构化世界、扮演生成式导演——只在你交互时推进世界、为每个角色维护主观认知、把控戏剧节奏、并在创作者设定的框架内自主引入新角色/事件。

> 核心意象：一扇可以通往任何世界的门——而那个世界，认得你。命名灵感同时致敬哆啦A梦的任意门。
