# 浮生 / The Reveries

一个移动优先的纯网页 **AI 活体文字世界**：像刷抖音一样竖滑发现"世界"，滑进一个就 fork 成你私有的实例长驻沉浸；一个 **God 引擎**维护结构化世界、扮演生成式导演——只在你交互时推进世界、为每个角色维护主观认知、把控戏剧节奏、并在创作者设定的框架内自主引入新角色/事件。

> 命名三重契合：「浮生若梦」的意境；《西部世界》里让接待员获得记忆/觉醒的核心更新 "the reveries"；以及 Stanford Generative Agents 代码库的内部代号 `reverie`。
