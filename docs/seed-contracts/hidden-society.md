# 夜庭 · 你欠下的第一笔人情

- **IP 路线**:结构化致敬(Vampire: the Masquerade / The Dresden Files;扒掉商标的
  原创隐秘社会)
- **炫的引擎强项**:派系权谋 · lore 按需展开(`establishLore` + 递归级联)
- **漏斗证明**:加深——「lore 一层层结晶为 canon」

> 为什么这个原型:它把 **lore 的递归展开**放正中央——问到「Accord」会牵出「三方」
> 「罚则」「旧战争」,知识图谱式按需结晶。也示范 `narration` 的「**帷幕失真**」:对
> 凡常视角默认隐去超自然,随玩家被引入而揭开,**失真是 in-world 的帷幕规则**。

## 1. 门卡(cold-open)

- **门名**:夜庭 · 你欠下的第一笔人情
- **冷开场**:你以为那只是一次走运的交易。直到午夜,一个西装革履的人坐到你对面,
  说你三天前签字的合同用的是夜庭的墨水——而夜庭里,人情是要拿命还的。
- **调性 / 烈度**:都市异闻 · 派系权谋 · 暗债 / 中
- **未决张力**:你欠了一笔自己都不知道规矩的债。
- **开门动作**:问对面这人到底是什么 / 低头看合同上正在渗血的签名

## 2. 硬规则 WorldRules

- **physics**:凡常物理 + **隐秘超自然**(吸血、术法、精怪契约——各有硬规矩与代价)。
- **socialOrder**:**三方停战协议(Accord)**是最高法;**帷幕(Masquerade)**:不可对
  凡人暴露;**恩义/契约是硬通货**,欠债要还、毁约有罚。
- **magicTech**:现代都市 + 隐秘体系;超自然**有代价、有规矩**(不是无限魔法)。
- **redLines**:平台基线;超自然**受规矩约束**(红线:不可凭空违约、破帷幕不可无代价)。
- **narration**:**忠实,但对凡常视角覆「帷幕」**——默认隐去超自然真相,随玩家被引入/
  获得「视界」而逐层揭开。失真是 **in-world 的帷幕规则**,**中枢始终持完整超自然真相**。
- **ruleSkills**:**恩义/契约账**——谁欠谁、债的性质与到期罚则(结构化 favor economy,
  作为 canon 强制;违约触发确定性后果)。轻量,味道独特。

## 3. 调性引力 tonalGravity

暗债与权衡。倾向把每次「帮助」都变成一笔将来要还的账;倾向让三方的猜忌一点点显形。

## 4. 开场局部 openingLocality

- **首场景**:午夜的**酒吧雅座**,西装人坐下。
- **玩家起点**:三天前无意签了夜庭合同的凡人(你为何走运到能签?你的秘密=你身上有
  某种他们想要的东西)。`condition`:签名处隐隐发烫;开始看得见一点别人看不见的东西。
- **在场可交互实体**:
  - **角色**:债主代理 vane、引荐者 lysa、凡人猎人 cole、术士代表 orrin、精怪代表
    (+ 酒客 ambient)。
  - **物件**:**渗血的合同**(契约物;动它=违约后果)、酒吧后门(`gated` 通夜庭,需
    「认领」才开=社会 `locked`)、一枚旧债**硬币**(符号)。
- **即时张力**:这债是谁的、要还什么?

## 6. 压力线 pressureLines(2–3,半隐)

1. **派系压力 · Accord 将裂**(随你卷入显形;迹象:某方挑衅、帷幕被撕开的小事故、
   代表的焦躁)。
2. **角色压力 · 你那笔债到期**(`ruleSkills` 的债期逼近,掮客逼债)。
3. **谜题压力 · 你身上「他们想要的东西」**(canon 锁定)。

## 7. 正典账本 canonLedger(**中枢持有,角色只有部分视角**)

- 你签的债**已被转手给精怪方**,抵押的是你身上某物。**锁定**:你不自知地带着某个旧
  血脉/一件信物。
- Accord 因一桩**未决的旧命案**正在崩解(锁定真相);三方各执一词。
- **完整超自然真相**(中枢持有;凡常视角被帷幕挡住,看不见)。
  > 信息差:vane 以为债还在自己手、lysa 想拿你翻她的债、cole 当你是怪物、orrin 怕
  协议崩、精怪盯着你的血脉——**各看一角,真相在中枢**。

## 8. 展开文法 expansionGrammar

- **新地盘**:三方领地随引入材化。
- **新角色**:持久才 `establishCharacter`(如另一脉的执法者),从夜庭自身出现。
- **新 lore**:大量靠 `establishLore` + **递归级联**(本世界主炫点)。
