# 灰汐山庄 · 退潮前七小时

- **IP 路线**:结构化致敬(Agatha Christie 封闭谜案 + 潮汐孤岛闭环;扒掉商标的原创世界)
- **炫的引擎强项**:见证作用域 · 秘密 · 信息差 · 社会因果
- **漏斗证明**:头十分钟——「有人**误记/误判**你」「一条本地后果落库」

> 为什么这个原型先写:它把引擎最难复制的东西(**结构性信息差**)放在正中央——
> 八个人共处一室,每人只知道自己见证的一角,真相在中枢。它也示范 `narration` 字段
> **必须**钉死在「忠实」端(公平推理),以及 `ruleSkills` 在社会世界**可空**。

---

## 1. 门卡(cold-open)

- **门名**:灰汐山庄 · 退潮前七小时
- **冷开场**:渡工把缆绳一松,海雾就合上了身后的路。客厅地毯上,老庄主的茶杯翻
  倒着,人没了气息——而潮水,要七个钟头后才退。
- **调性 / 烈度**:悬疑 · 封闭 · 猜疑 / 中高
- **未决张力**:凶手就在这屋里的八个人之中,退潮前谁都走不了。
- **开门动作**:蹲下查看那只翻倒的茶杯 / 抬头环视客厅里每一张脸

## 2. 硬规则 WorldRules

- **physics**:寻常物理。**潮汐时钟**是硬空间约束——通往大陆的堤道被涨潮淹没
  (`gates: 堤道→大陆`,`locked: true`),船已离开,七小时(游戏时间)后退潮才开。
- **socialOrder**:没落贵族的私人山庄;主—客—仆三级身份;**名誉与丑闻是硬通货**,
  一句指认能改变一个人的处境。
- **magicTech**:1920s 风。无超自然。电话线在暴风雨里断了——**无即时外援**。
- **redLines**:平台基线 + 本世界额外:**不引入任何超自然解释**(公平推理,凶手是
  这八人之一,且线索都在场内)。
- **narration**:**忠实(faithful),钉死。** 这是 fair-play whodunit——旁白不得
  欺骗玩家任何**硬事实**(谁在场、物证状态、时间线)。失真会摧毁推理的公平性。
  *(角色仍可撒谎——那是角色声部,会被 Reactor 路由成「谎言/信念」,与旁白忠实正交。)*
- **ruleSkills**:**基本为空(社会世界)。** 可选一条极轻的 `alibi-consistency`
  critic:把角色声称的不在场证明与 `deltaLog` 的在场记录比对,供 Director 发现矛盾
  (不是数值仿真,只是一致性校验)。——示范该栏对社会世界的最小形态。

## 3. 调性引力 tonalGravity

缓慢收紧的猜疑。每个人都有不可告人的小秘密,但**只有一个秘密是杀人**。世界倾向把
对话推向试探、伪证、互相指认;倾向让退潮的水位声成为背景压力。

## 4. 开场局部 openingLocality

- **首场景**:**客厅**(壁炉将熄)。八人围着倒地的老庄主,茶杯翻在地毯上。
- **玩家起点**:你是赶上**最后一班渡轮**的不速之客。`condition`:浑身海雾的湿气;
  口袋里有一封**把你引来的匿名信**(钩子,内容留白给玩家发现)。
  > 玩家是**唯一从世界外推门进来的人**;其余八人都是山庄native。你为何被引来,是
  > 你自己的秘密。
- **在场可交互实体(开场即可 engage)**:
  - **角色**:管家老埃德加、遗孀薇拉、长子朱利安、私生女萝丝、家庭医生哈洛
    (+ 几名仆役/宾客默认 **ambient**,被 engage 才结晶)。每回合 Director 选 ≤4 个
    active。
  - **物件**:**翻倒的茶杯**(物证;`portable:true` 但**拿走它=动证据→社会后果**,
    物主关系账本受损)、**壁炉**、**书房门**(`gates: 书房`,`locked:true`)、
    墙上**全家福**(符号:看谁被裁掉了)。
- **即时张力**:老庄主到底是中毒,还是心脏病?**那只茶杯会说话。**

## 6. 压力线 pressureLines(2–3 条,半隐)

1. **世界压力 · 潮水时钟**:七小时后堤道开——届时凶手可能脱身、或外界将至。
   *diegetic 迹象*:窗外水位、远处雾笛、壁炉的余烬。*公平性*:水位人人可见,是公平的钟。
2. **角色压力 · 遗嘱之争**:多人当晚都动过书房;真相曝光→继承权翻盘。
   *迹象*:书房门的划痕、某人袖口的灰、欲言又止。
3. **谜题压力 · 真正死因**:茶杯 → 验尸 → 医生的谎,层层逼近。**真相随玩家逼近而显形**;
   一旦触及即进 active canon(公平性:无迹象不重罚,有迹象而忽视则后果加重)。

## 7. 正典账本 canonLedger(**中枢持有,角色只有部分视角**)

- 老庄主死于**洋地黄(digitalis)中毒**,**非**心脏病。
- **答案(锁定,绝不进任何角色初始知识)**:**萝丝**把过量洋地黄混进父亲的「安眠茶」,
  本意是让父亲昏睡、好逼他当众认她——**过量致死,半蓄意的悲剧**。**薇拉端了茶,但不
  知情。**
  > 这把信息差做到极致:薇拉以为自己**可能误杀**、朱利安咬定是**薇拉**、埃德加疑
  **遗孀**、哈洛知道是**洋地黄**却怕暴露自己开错药……**没有任何一个人知道全貌。**
  真相只在中枢,随玩家的盘问、验尸、搜证一寸寸显形。
- 堤道七小时后退潮(世界硬事实)。

## 8. 展开文法 expansionGrammar

- **新地点**:沿山庄合理延展(酒窖、阁楼、家庙),按玩家走向**材化**。
- **新角色**:仅当剧情需要**持久**人物时 `establishCharacter`(如「当年被遣散的旧
  仆」),从世界**自身**egress 出现(走廊、里屋、堤道方向)——**绝不**框成玩家的门。
- **新 lore**:遗嘱条款、家族旧案按需 `establishLore` 结晶为永久 canon,永不自相矛盾。

## 自检:它兑现了什么

- **见证作用域**:朱利安在回廊瞥见萝丝 → 只有他有这条观察;玩家要套出来。
- **秘密 / 信息差**:五个人五套私有信念,**全与中枢真相不同**——戏剧反讽天然产生。
- **社会因果**:动茶杯=动证据→物主记恨;一次公开指认改变某人处境(关系账本)。
- **narration 字段**:被本世界**钉死在忠实**——示范它是 per-world 设定,不是开关。
- **ruleSkills 字段**:本世界近乎空——示范社会世界不需确定性计算(留给 `one-night-dungeon` 炫)。
