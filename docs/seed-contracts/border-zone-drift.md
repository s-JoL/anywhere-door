# 灰带 · 没有回头路的第三道门

- **IP 路线**:结构化致敬(STALKER / 魂系;扒掉商标的原创禁区)
- **炫的引擎强项**:按注意力材化(progressive unfolding)· ambient lore(环境叙事)
- **漏斗证明**:跨阈——「推门即入的氛围」

> 为什么这个原型:它把**按注意力材化**放正中央——不预生成大地图,凑近才长出地点与
> 异常。也示范 `narration` 的「**惜字不解释**」(魂系留白):世界只呈现痕迹,不主动
> 解说,ambient lore 靠玩家拼。

---

## 1. 门卡(cold-open)

- **门名**:灰带 · 没有回头路的第三道门
- **冷开场**:引路人收了钱,在第三道门前停下,说他只送到这。门后是灰带——天是铁锈色
  的,空气在某些地方会扭曲,而所有进去的人,都是为了同一样东西回不来。
- **调性 / 烈度**:废土探索 · 孤独 · 环境之惧 / 中
- **未决张力**:你为之而来的东西在灰带最深处;进去的人都没回来。
- **开门动作**:拾起引路人留下的螺母(探异常用)/ 迈过第三道门

## 2. 硬规则 WorldRules

- **physics**:**被灾变改写的物理**——某些地点有「异常」(扭曲/灼烧/失重区),可探测、
  可致命;空间可游走但危险地点 `gates` 通行。
- **socialOrder**:几乎无社会;偶遇的漫游者各为其利;**装备与情报是硬通货**。
- **magicTech**:灾变后科技废墟 + 无解释的异常(归为「灾变物理」,不归魔法)。
- **redLines**:平台基线;异常**有规律可循**(红线:不搞无规律的纯随机致死=不公平)。
- **narration**:**忠实,但惜字、不解释**(魂系留白)——世界只呈现现象与痕迹,**不
  主动解说**;ambient lore 靠玩家从环境拼。示范 narration 规则可定**密度/曝光度**
  (克制、环境化),仍忠实。
- **ruleSkills**:**异常逻辑**——异常类型 + 触发条件 + 探测信号(螺母/计数器读数);
  踩错=确定性后果。**有规律 = 公平**。轻量但真用上。

## 3. 调性引力 tonalGravity

孤独的逼近。倾向让环境痕迹替代对白;倾向把每一步都变成风险评估。

## 4. 开场局部 openingLocality

- **首场景**:**第三道门前**,引路人转身离开。
- **玩家起点**:为「那样东西」深入的漫游者(为何非要它?你的秘密)。`condition`:旧
  防护服、一只盖革计数器、半盒火柴;一张手绘的残破地图。
- **在场可交互实体**(少而精):
  - **角色**:即将离开的引路人、远处一个不动的**身影**、深处据点的**隐居者**——多为
    ambient/环境。
  - **物件**:**螺母**(探异常)、**盖革计数器**(读数)、**扭曲的空气**(异常,`gates`)、
    锈门、墙上前人刻的**记号**(ambient lore)。
- **即时张力**:门后第一段路,**哪里能踩?**

## 6. 压力线 pressureLines(2–3,半隐)

1. **世界压力 · 灰带本身**(异常随深入加密;迹象:计数器、扭曲、动物尸体的朝向)。
   *公平*:有信号可读。
2. **谜题压力 · 「那样东西」是什么、值不值**(随逼近显形;canon 锁定)。
3. **角色压力 · 隐居者的阻拦 / wraith 的镜像**(你越深,越照见自己的执念)。

## 7. 正典账本 canonLedger(**中枢持有,玩家只能从信号推**)

- 「那样东西」= **锁定**:一枚能实现一个愿望的**灾变核心**,代价是把你也变成灰带的
  一部分——**这正是 wraith 的来历**(上一个为它而来的人)。
- 灾变**原点真相** + **异常的规律**(中枢持完整规则,玩家只能从螺母/计数器推)。
- sma 是更早一批失败者的幸存者,知道代价,故阻拦。

## 8. 展开文法 expansionGrammar

- **新地点**:灰带**按玩家走向材化**(本世界主炫点:凑近才长出地图与异常)。
- **新角色**:极少,持久才 `establishCharacter`;从灰带自身出现。
- **新 lore**:刻字/日记/原点按需 `establishLore`。

## 自检:它兑现了什么

- **按注意力材化**:深入才长出地点与异常——跨阈/探索的体感。
- **ambient lore**:环境痕迹(尸体朝向、刻字、日记)替代解说。
- **narration 惜字不解释**:世界不主动说明,留白即风格,仍忠实。
- **ruleSkills 异常逻辑**:有规律可探 = 死得明白、公平,不是随机劝退。
