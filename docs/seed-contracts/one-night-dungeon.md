# 黑根地牢 · 天亮前的一趟

- **IP 路线**:平台原创(轻量 TRPG 一夜本)
- **炫的引擎强项**:**agent 化世界**——战斗/检定/资源/限时**确定性计算**
- **漏斗证明**:证明「game-y 也机械成立」(死亡不靠 LLM 心软)

> 为什么这个原型:它是**验证第二轮升级最值钱部分**的活样板——`ruleSkills` 核心必填,
> agent 化 Director 把战斗/检定/资源/限时**确定性运算**成 delta、过闸门落库。与社会
> 世界(庄园/小镇 `ruleSkills` 空)形成对照,证明同一引擎能从纯叙事一路覆盖到精确裁定。

---

## 1. 门卡(cold-open)

- **门名**:黑根地牢 · 天亮前的一趟
- **冷开场**:雇你的人只给一夜:子时下去,卯时前带着「心石」上来,多耽一刻,闸门
  落下就再上不来。四个临时凑的同伴,一盏快烧完的火把,和一段没人活着走完过的下行
  阶梯。
- **调性 / 烈度**:地牢探险 · 限时 · 硬核检定 / 中高
- **未决张力**:一夜、一支生队伍、一座没人活着走通的地牢。
- **开门动作**:检查队伍的装备与状态 / 点起火把迈下第一级台阶

## 2. 硬规则 WorldRules

- **physics**:剑与魔法的地牢;**确定性规则裁定**(检定/战斗/伤害/资源由 `ruleSkills`
  计算);空间下行,密室/机关 `gates`+`locked`。
- **socialOrder**:临时雇佣小队;**信任与分赃**;队友各有底线。
- **magicTech**:低魔奇幻;魔法**按规则消耗资源**。
- **redLines**:平台基线;**死亡有真实后果**(HP 归零=确定性结果,不靠 LLM 心软)。
- **narration**:**忠实,把确定性结算散文化**;默认不显数字,**Studio/Inspector 可看
  掷骰回执**(透明)。
- **ruleSkills**:**核心必填 `trpg-core`**——属性(力/敏/智/魅)、HP/护甲、`d20+调整
  vs DC` 检定、伤害骰、资源(火把回合数/药水/绳)、遭遇与陷阱表、**限时回合钟**(卯时
  闸门)。agent 化 Director 每次冲突/检定**确定性运算 → 提议 delta**(改 HP/资源/状态),
  过闸门落库。**本样板主炫点。**

## 3. 调性引力 tonalGravity

限时高压。倾向把每个选择变成风险/资源取舍;倾向让骰子的残酷与队友的算计显形。

## 4. 开场局部 openingLocality

- **首场景**:**地牢入口**,子时,闸门半开,火把待点。
- **玩家起点**:受雇的探险者(你为何接这趟死活?你的秘密)。`condition`:HP 满、基础
  装备、属性已定(随机/可选);倒计时:**卯时**(约 N 回合)。
- **在场可交互实体**:
  - **角色**:战士 brak、盗贼 sly、法师 wynn、动机可疑的向导 fern(+ 地牢生物按遭遇表
    ambient,触发才成实体)。
  - **物件**:**火把**(资源,回合递减=照明/视野)、**心石**(目标,深处)、机关门
    (`locked`/`gated`,需检定/钥匙)、宝箱(检定开,可能有陷阱)。
- **即时张力**:第一道岔路 + 火把只够一段时间。

## 6. 压力线 pressureLines(2–3,半隐)

1. **世界压力 · 限时闸门**(`ruleSkills` 回合钟驱动;迹象:火把、远处水滴计数、向导
   催促)。*公平*:回合数可知。
2. **角色压力 · 向导的隐藏指令 / 盗贼的独吞心**(随深入显形)。
3. **谜题压力 · 心石到底是什么、雇主为何要它**(canon 锁定)。

## 7. 正典账本 canonLedger(**中枢/`ruleSkills` 持有,角色只有部分视角**)

- 心石 = **锁定**:封印地牢深处某物的**钥石**,取走会**放出**那东西——雇主真正想要的
  正是「放出来」这件事;向导 fern 受命**确保心石被取走**。
- 上趟队伍**死于此真相**(brak 的兄弟即其一)。
- **TRPG 数值真相**(中枢/`ruleSkills` 持确定性状态:各人 HP/资源/回合数)。
  > 信息差:brak 疑向导、sly 想独吞、wynn 怕撑不住、fern 在执行黑指令——**各看一角**。

## 自检:它兑现了什么

- **agent 化世界炫点**:战斗/检定/资源/限时全由 `ruleSkills` **确定性计算**,经闸门
  落 delta——**证明 game-y 也机械成立**(死亡不靠 LLM 心软)。
- **narration**:把数字散文化,Studio/Inspector 可看掷骰回执(透明)。
- **对照**:与庄园/小镇(`ruleSkills` 空)同台,证明**同一引擎从纯叙事覆盖到精确裁定**。
- **角色 POV 仍在**:向导的黑指令、盗贼的独吞,信息差照样驱动戏。
