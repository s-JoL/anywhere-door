# 运河区 · 雨夜值班表第一行

- **IP 路线**:结构化致敬(Disco Elysium / True Detective S1;扒掉商标的原创城区)
- **炫的引擎强项**:压力线 · 道德分歧(不同角色用不同价值判同一行为)· 角色 POV
- **漏斗证明**:头十分钟——「一条本地后果落库」

> 为什么这个原型:它把**道德分歧**放正中央——同一个「压案」决定,五个人用五套价值
> 去判(自保 / 还债 / 背叛 / 活命 / 公道)。也示范 `narration` 规则可塑**语气/声音**
> (Disco Elysium 式内心声)而**不歪曲硬事实**。

## 1. 门卡(cold-open)

- **门名**:运河区 · 雨夜值班表第一行
- **冷开场**:你调来的第一夜,雨没停过。运河边浮起一个人,口袋里是半页港务局的分赃
  名单——而你这分局里每个人的名字,可能都在另外半页上。
- **调性 / 烈度**:黑色 · 制度腐败 · 道德泥潭 / 中高
- **未决张力**:案子牵出的名单,也许就压在你同事的抽屉里。
- **开门动作**:翻死者口袋里那半页名单 / 看一眼值班室里此刻盯着你的人

## 2. 硬规则 WorldRules

- **physics**:寻常物理;雨夜运河区(分局、码头、酒馆、停尸间可达);档案室 `locked`。
- **socialOrder**:港务工会—分局—航运公司**三方分赃**的潜规则;**沉默是规矩**,捅破
  =与全街为敌;**人情与把柄是硬通货**。
- **magicTech**:近当代,无超自然。
- **redLines**:平台基线;不引入超自然——**腐败是人的腐败**。
- **narration**:**忠实事实,语气受你当下状态调制**(疲惫/酒意/恐惧给同一场景不同
  笔触)。示范 narration 规则可塑**声音/语气**(内心声式),而硬事实(谁在场、物证
  状态、时间线)**仍忠实**。
- **ruleSkills**:**近空**。街区对你的「信任/怀疑」由既有**社会关系账本**承载,不需
  自定义计算。

## 3. 调性引力 tonalGravity

缓慢下沉的道德泥潭。倾向把每个选择逼成「你站哪边」;倾向让沉默与共谋一点点显形。

- **玩家起点**:今夜刚调来的警探(为何被发配到这种地方?你的秘密)。`condition`:

- **在场可交互实体**:
  - **角色**:老警司 boldt、搭档 nessa、验尸官 calder、酒馆老板 ido、死者姐姐 wren

    墙上分局合影(符号:谁笑得勉强)。
- **即时张力**:死者是淹死还是抛尸?**验尸官的脸告诉你别问。**

## 6. 压力线 pressureLines(2–3,半隐)

1. **制度压力 · 分赃链曝光**(随你查显形;迹象:同事回避、被换掉的证物、半夜来电)。
   *公平*:你越捅,街区越冷。
2. **角色压力 · 搭档的人情债**(nessa 当晚在场,你迟早撞见)。
3. **谜题压力 · 真死因与名单全貌**(canon 锁定:谁杀、为何)。

## 7. 正典账本 canonLedger(**中枢持有,角色只有部分视角**)

- 死者**他杀**,非溺亡。**锁定**:航运公司的人灭口,老警司 boldt **默许压案**;搭档
  nessa 当晚撞见但收了沉默的人情,**没动手**。
- **名单另半页**在酒馆老板 ido 处,上面有 boldt、有公司、有工会。
- 各人只知一角:boldt 怕被查、nessa 怕暴露在场、calder 怕封口钱、wren 不信任何警察。
  > **道德分歧的种子**:同一个「压案」决定,boldt 看是**自保**、nessa 看是**还债**、
  > wren 看是**背叛**、calder 看是**活命**、ido 看是**生意**——五套价值判同一件事。

## 自检:它兑现了什么

- **道德分歧**:五套价值判同一个「压案」——同一行为在不同角色眼里是自保/还债/背叛/
  活命/生意。
- **压力线 diegetic**:回避、换证、来电,不开任务日志。
- **角色 POV**:nessa 的在场只有她自己知道,玩家要套。
- **narration 语气调制**:疲惫/酒意给同一场景不同笔触,**硬事实不动**。
